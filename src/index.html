Game of life - an excercise project using Phaser

An excercise project

This was an exercise project at Turku University software engineering class. I have very limited experience with html, css or javascript but I think this turned out nicely.

Original requirements:

  • Scrolling text
  • Some effect (this is using Tweens to fade out the dead cells)
  • Music - Eric Skiff - Underclocked (underunderclocked mix)

What?

This simulation uses classic Game of Life rules. Additionally, if there are less than 15 new cells born in a generation a counter is incremented and game resets after certain amount is reached. This prevents some cases where it gets stuck oscillating some uninteresting small formations.

Sliders above are used to control a particle effect when a cell dies. I did some experimenting but I did not like it too much. However I left it there so feel free to test. Also source code is here (not yet though).
